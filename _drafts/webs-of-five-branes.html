Let us consider a configuration in type IIB string with various kinds of five-branes. The first kind is the $D5$ brane, whose tension is (see this previous post for the general formula) :
\begin{equation}
\tau_{D5} = \frac{1}{g_s (2 \pi )^5 \alpha ' ^3} \, .
\end{equation}
We would like to see how this transforms under $S$-duality, which changes $g_s \mapsto 1/g_s$. For this, we have to know the transformation rule for $\aplha '$. One way of finding it is to look at the strings: the tension of a fundamental string is $\tau_{F1} = \frac{1}{2 \pi \alpha '}$, and under $S$-duality it becomes a $D1$ brane with tension $\tau_{D1} = \frac{1}{2 \pi \alpha ' g_s}$, so we have the rule
\begin{equation}
S :
\begin{cases}
g_s \mapsto \frac{1}{g_s} \\
\alpha ' \mapsto \alpha 'g_s \, .
\end{cases}
\end{equation}
Using this rule, we find the tension of the $NS5$ brane, which is the dual of the $D5$:
\begin{equation}
\tau_{NS5} = \frac{1}{g_s^2 (2 \pi )^5 \alpha ' ^3} \, .
\end{equation}

String networks

It is not entirely obvious that we can combine a $D1$ and a $F1$ into a BPS object. Indeed, the BPS bound (computed from the supersymmetry algebra) for a one-dimensional object with $p$ units of fundamental string charge and $q$ units of $D1$ charge is $\tau \geq \tau^{(p,q)}_1 = \sqrt{(p \tau_{F1})^2 + (q \tau_{D1})^2}$. Putting just one $D1$ and one $F1$ on top of each other produces an object of tension $\tau_{F1} + \tau_{D1}$, which doesn't saturate the BPS bound. But this is not the stable situation: the fundamental string will break, with ends appearing on the $D1$, and finally dissolves, leaving only flux in the $D1$. Then one can compute that the tension of the resulting object saturates the BPS bound! And one can do the same for any $(p,q)$. Note that we can write
\begin{equation}
\tau^{(p,q)}_1 = \frac{1}{2 \pi \alpha '} \sqrt{p^2 + (q /g_s)^2} = \tau_{F1} |p+\tau q| \, ,
\end{equation}
with $\tau = i / g_s$. Finally, we can consider string junctions. In order to preserve supersymmetry, they need to satisfy two conditions:

  1. The sum of the charges must vanish at each junction
  2. The string with charge $(p,q)$ must be pointing in the direction of $p +Â  \tau q$ in the plane of the junction.

We can then take the $S$-dual of this construction, and end up with a brane web. The branes are extended in dimensions $0,1,2,3,4$ and they form a network in the plane $5,6$, submitted to the same conditions as the corresponding strings.

Brane webs

If we consider a web of five-branes, then its world-volume is a five-dimensional $\mathcal{N}=1$ supersymmetric QFT. The original paper by Aharony, Hanany and Kol, Webs of (p,q) 5-branes, Five Dimensional Field Theories and
Grid Diagrams, then gives a dictionary between the $5d$ point of view and the web of branes point of view.
